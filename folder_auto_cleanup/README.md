# Folder auto-cleanup
A small Bash script that auto-sorts files from the requested folder into subfolders (e.g., "Images/", "PDF", ...) based on the file extension.

1. **Download the script**  
   Download the folder_auto_cleanup.sh file and make it executable.

2. **Download the additional file**
   Download the `categories.sh` file and save it in the same folder as `folder_auto_cleanup.sh`.

3. **Run the script**  
   Run the script and add the name of the folder that you want to sort as an argument.
